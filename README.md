## Description
    npm package for adding border-radius to your elements
## Installation
    `npm i ktsradiuswizard --save`
    ```

    import {radisuwizard} from 'ktsradiuswizard';
    radiuswizard({
        circle:false
    })

    ```
### Options
    radiuswizard supports only one option
    -_circle_ - _boolean_ (Defaults to false)
